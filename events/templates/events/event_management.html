Event Management

Events

To create an event you will need a name, date start, date end, run time, short & long description and a hero & normal image. Also, please ensure your hero image is 1500x400 pixels and your normal image is 1000x800 pixels.

Showtime

Hopefully you have an event and venue in mind for your showtime. Otherwise, all you need is a date, time and a ticket price.

Venue

To create a Venue, ensure it has a name, capacity, description and image. you will also require a seat map. Ensure the image is 1500x1000 pixels. When creating a seat map, use 'g' to represent a seat, '_' to represent empty space and ensure that the same ammount of characters are used on each row which you can seperate with a comma. An example would be: ggg_ggg,ggg_ggg,ggg_ggg
